# Travel Planner

SPA-приложение для планирования путешествий и организации поездок в формате dashboard.

Проект реализуется как pet-project для практики frontend-разработки: построение SPA на React, работа с TypeScript, проектирование структуры приложения и реализация пользовательских сценариев.


## Цель проекта

Практика разработки интерфейсов и архитектуры frontend-приложений:

- построение SPA на React
- работа с TypeScript в реальном проекте
- проектирование структуры приложения
- работа с пользовательскими формами и состояниями
- реализация типичных UI-сценариев dashboard-приложений


## Основной функционал (MVP)

- Базовая авторизация пользователя
- Управление поездками (CRUD)
- Просмотр и редактирование деталей поездки
- Планирование задач поездки (Checklist)
- Заметки по поездке


## Планируемые улучшения

- Карта для визуализации маршрутов
- Погода по локации поездки
- Backend на Supabase (данные и авторизация)
- AI-помощник для генерации маршрутов и рекомендаций


## Tech Stack

- React
- TypeScript
- Vite
- React Router (SPA navigation)
- SCSS Modules (modular styling)


## Примечание

Проект находится в активной разработке. Структура и функциональность будут расширяться по мере реализации.
